## Установка

Чтобы установить это приложение на свой рабочий стол, выполните следующие действия:

1. Загрузите последнюю версию приложения со страницы релизов на Github.
2. Установите приложение на рабочий стол, запустив загруженный файл установщика.
3. После завершения установки запустите приложение, чтобы начать писать markdown
