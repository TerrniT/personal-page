## Fokasu (焦点, 中心, 病巣, Фокус)

Приложение Помодоро, которое поможет вам сосредоточиться и повысить продуктивность с помощью техники Помодоро. Fokasu предлагает интуитивно понятный интерфейс и настраиваемые параметры, чтобы помочь вам эффективно управлять рабочими и перерывами.

## Особенности

- **Настраиваемые параметры**: Регулируйте длительность работы, длительность перерывов и количество циклов Помодоро в соответствии с вашим рабочим процессом.
- **Отслеживание прогресса**: Отслеживайте завершенные сессии Помодоро и текущие серии.
- **Чистый интерфейс**: Минималистичный дизайн, который позволяет сосредоточиться на задачах без отвлечений.
- **Таймер и процент оставшегося времени**: Визуальный таймер обратного отсчета с указанием оставшегося процента для лучшего управления временем.

## Установка

Чтобы запустить приложение Fokasu локально, выполните следующие шаги:

1. **Клонировать репозиторий**:
   ```bash
   git clone https://github.com/terrnit/fokasu.git
   cd fokasu
   ```

2. **Установить зависимости**:
   Убедитесь, что у вас установлены [Node.js](https://nodejs.org/), [npm](https://www.npmjs.com/) и [rustup](https://tauri.app/start/prerequisites/#rust). Затем выполните:
   ```bash
   npm install
   ```

3. **Запустите сервер разработки**:
   ```bash
   npm run dev
   ```

## Использование

1. **Установите ваши предпочтения**: Перейдите в настройки, чтобы настроить длительность работы, длительность перерывов и количество циклов.
2. **Запустите таймер**: Нажмите кнопку "Start", чтобы начать сессию Помодоро.
3. **Отслеживайте свой прогресс**: Визуально наблюдайте за оставшимся временем и оставшимся процентом.
4. **Делайте перерывы**: После каждой сессии Помодоро делайте перерыв, чтобы восстановить силы.
5. **Проверьте свой прогресс**: Проверьте завершенные сессии и серии, чтобы оценить продуктивность.

## Настройка

- **Изменение настроек**: Вы можете изменить длительность работы и длительность перерывов прямо в настройках приложения.
- **Добавление новых функций**: Приветствуются предложения по улучшению приложения! Не стесняйтесь вносить изменения через PR (Pull Request).
