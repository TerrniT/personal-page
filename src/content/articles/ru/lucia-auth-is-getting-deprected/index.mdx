> [pilcrowonpaper](https://github.com/pilcrowonpaper) - создатель Lucia Auth  
   
Автор признаёт, что текущая реализация библиотеки Lucia не соответствует потребностям разработчиков, что привело к решению о прекращении поддержки в начале следующего года.  
Новое направление — создание open-source ресурса, который научит реализации аутентификации на JavaScript с нуля. Акцент смещается с готового пакета на образовательный контент. Сложные адаптеры баз данных стали источником проблем, и новый подход упростит поддержку.  

Хотя Lucia могла бы развиться в полноценный фреймворк вроде Auth.js, автор предпочитает сохранить её как отдельный пакет. Версия 3 будет поддерживаться ещё 6 месяцев (до марта 2025), чтобы дать пользователям время на миграцию без изменения схемы БД.  
Сейчас Lucia можно использовать как руководство по созданию аутентификации с нуля.  

## Мнение  
Главная проблема — попытка охватить все возможные сценарии использования. Для сложных кейсов лучше создавать кастомные решения.  
Для базовых задач (OAuth-провайдеры, `validateRequest()`, управление сессиями) Lucia остаётся отличным лёгким решением. Она даёт ключевые функции без лишней сложности.  
Огромная благодарность @pilcrowonpaper за работу над библиотекой и материалами!
